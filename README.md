# Задание 1: построение ML-системы обработки потоковых данных

**Проект:** Система обработки потоковых данных, контроля их качества и непрерывного обучения ML-моделей. 

**Набор данных:** Rain in Australia (Predict next-day rain in Australia).

## Архитектура
Конвейер реализован на базовых библиотеках Python (`pandas`, `scikit-learn`).

Архитектура включает:
1. `data_collection` - сбор потоковых батчей и расчет метапараметров.
2. `data_quality` - проверка данных через 5 логических правил (Apriori) и очистка пропусков.
3. `model_training` - сборка Pipeline и обучение дерева решений и нейронной сети.
4. `model_validation` - применение TimeSeriesSplit для валидации моделей и формирование хранилища версий.
5. `model_inference` - применение лучшей модели (сериализированной через `pickle`) к новым данным.
6. `report_generator` - парсинг логов системы для формирования Summary-отчета.
